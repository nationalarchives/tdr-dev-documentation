# TDR Add Grafana User

TDR uses a AWS managed Grafana instance.

To add a new user:
1. Go to the TDR management account AWS console;
2. Navigate to the Amazon Grafana service;
3. Go to the Grafana instance workspace;
4. Under the `Authentication` tab, select the `Configure users and user groups` option
5. Add the new user with the relevant permissions
